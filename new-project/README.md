# New Project

这是一个新建的项目仓库模板，可作为后续业务开发的起点。

## 快速开始

```bash
npm install
npm start
```

## 目录结构

```text
new-project/
├── src/
│   └── index.js
├── .gitignore
├── package.json
└── README.md
```

## 可用脚本

- `npm start`：运行示例入口。
- `npm test`：运行基础检查。
